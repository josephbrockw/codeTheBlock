Frontend Web Development

This is where recommend most of our students start. By the end of this program, you'll be able to build your own basic websites and have the skillset to contribute to larger projects. We believe that frontend web development is a great place to begin because it’s very visual but if you aren’t interested, you can start with one of our programs with a little prework.

We currently only run this as an in-person program but are working to add digital access soon

App Development

Our app development program is exclusively focused on React Native because it is one of the most dynamic and cost effective ways to build native apps. React Native is JavaScript based, which makes it easier to learn but still has most of the functionality you'll ever need. In this program we start to get into some backend languages and technologies as well.

We currently only run this as an in-person program but are working to add digital access soon
