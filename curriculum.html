Frontend Web Development

Frontend Web Development is a good way for students to ease into coding. By the end of this program, you'll be able to build your own basic websites and have the skillset to contribute to larger projects. We believe that frontend web development is a great place to begin because it’s very visual.

App Development

Our app development program is exclusively focused on React Native because we believe it is one of the most dynamic and cost effective ways to build native apps. React Native is JavaScript based, which makes it easier to learn but still has most of the functionality you'll ever need. In this program we start to get into some backend languages and technologies as well.

Introduction to Software Engineering

Our Introduction to Software Engineering program is designed to teach students the basics of a programming language as efficiently as possible so that they can start building real projects early on in their learning. This allows students to get a better feel for how to work in a real team development environment and build a portfolio at the same time.

An area where a lot of programs leave students feeling unprepared is in ancillary development technologies. Most programs simply teach programming languages. We try to introduce as many supportive technologies as we can so that the transition into the workforce is as seemless as possible, reducing the stress and the feelings of being an imposter.
